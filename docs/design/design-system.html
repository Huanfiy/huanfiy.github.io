Huanfly 站点 · 设计架构可视化

「手绘绘本 × 罗小黑森林」主题（2026-07-02 重构）的模块组成、数据流与兼容边界。点击步骤或用 ←/→ 键切换视角；细节规范见同目录 design-system.md。

视觉签名元素

纸纹噪点（feTurbulence data URI）· wobble 不规则圆角四档 · 墨线描边 + 错位硬阴影 · 胶带贴纸 · 波浪线下划线 · 马克笔高亮 · Hero 三层山丘 + 小黑 SVG（眨眼 5 s / 摇尾 3.6 s / 抖耳 7 s）· 草地页脚。

动效与降级

Canvas 双层：漂浮萤火（z-index 5，10/20 个，随主题调亮度，标签页隐藏暂停）与点击灵气迸发（z-index 9950，18+8 粒子，冷却 90 ms）。prefers-reduced-motion 下萤火层不创建、迸发退化为单圆环、CSS 动画全部关闭。

z-index 秩序

5 萤火 → 99 移动端菜单 → 100 Header → 9950 点击迸发 → 9960 纸纹 → 9990 Toast → 9999 Lightbox。纸纹覆盖在多数内容之上（alpha ≤ 0.05），Toast 与 Lightbox 位于纸纹之上保持清晰。

主题机制

主题存储于 localStorage.theme，各页 <head> 内联脚本首帧前写入 <html data-theme> 防闪白；无存储值时跟随系统 prefers-color-scheme。深色 = 夜之森林：整组 Token 覆盖，山丘 / 小黑 / 页脚随 CSS 变量联动。
